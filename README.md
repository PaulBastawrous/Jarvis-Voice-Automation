# Jarvis Voice Automation

## Overview
Jarvis is a Python based voice assistant designed to automate real world workflows such as opening applications, navigating systems, and performing multi step tasks through voice commands.

## Features
Voice recognition using speech_recognition  
System automation using pyautogui  
Custom commands for navigation and task execution  
Fast execution of repetitive workflows  

## Technologies Used
Python  
speech_recognition  
pyautogui  
OS automation  


## How It Works
The assistant listens for a trigger word and processes commands using speech recognition.  
Once a command is identified, it executes pre programmed automation steps such as opening browsers, logging into systems, or navigating interfaces.
